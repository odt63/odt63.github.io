按位与和按位或

按位与 (&\And&) 是指在二进制下对两个数的每个数位分别进行与运算，例如：

5&7=(101)2&(111)2=(101)2=55 \And 7 = (101)_2 \And (111)_2 = (101)_2 =5 5&7=(101)2​&(111)2​=(101)2​=5

按位或 (∣|∣) 是指在二进制下对两个数的每个数位分别进行或运算，例如：

5 ∣ 6=(101)2 ∣ (110)2=(111)2=75 \ | \ 6 = (101)_2 \ | \ (110)_2 = (111)_2 =7 5 ∣ 6=(101)2​ ∣ (110)2​=(111)2​=7

注意：按位与（C++ 中为&）逻辑与（C++ 中为&&）、按位或（C++ 中为|）逻辑或（C++ 中为||）不同，请注意区别。

思路

这个题分三种情况。如果 m=nm=nm=n ，显然要操作的次数为 000 ，如果 n ∣ m=mn\ |\ m=mn ∣ m=m 或 n&m=mn \And m=mn&m=m ，那么要操作的次数为 111 ，否则要操作的次数为 222 。

为什么这样正确呢？

对于任意两个 nnn 和 mmm ，可以这么理解：先将 n&0n \And 0n&0 ，然后将 n ∣ mn \ | \ mn ∣ m 就可以得到 mmm ，最多只需要两步。

还有一种特殊情况，就是 nnn 中有的 111 ， mmm 中没有，或 mmm 中有的 111 ， nnn 中没有，其他部分完全相同时只需要 &\And& 或 ∣ m| \ m∣ m 就可以了，例如：

n&m=(1010101010101)2&(1010001010001)2=(1010001010001)2n \And m = (1010101010101)_2 \And (1010001010001)_2 = (1010001010001)_2 n&m=(1010101010101)2​&(1010001010001)2​=(1010001010001)2​

n ∣ m=(1010001010001)2 ∣ (1010101010101)2=(1010101010101)2n \ | \ m = (1010001010001)_2 \ | \ (1010101010101)_2 = (1010101010101)_2 n ∣ m=(1010001010001)2​ ∣ (1010101010101)2​=(1010101010101)2​

这样只需要一步就可以解决。
